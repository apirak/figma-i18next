# Figma i18next

## Development guide

_This plugin is built with
[Create Figma Plugin](https://yuanqing.github.io/create-figma-plugin/)._

### Build the plugin

To build the plugin:

```
$ npm run build
```

To watch for code changes and rebuild the plugin automatically:

```
$ npm run watch
```
